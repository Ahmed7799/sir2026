📋 Backend Setup Guide

  1. Open your Google Sheet → Extensions → Apps Script
  2. Delete existing code, paste the entire Code.gs file
  3. Click Deploy → New deployment → Web App
  4. Execute as: Me  |  Access: Anyone
  5. Click Authorize access when prompted → allow
  6. Copy the Web App URL → paste above → Connect

⚠️ If you update Code.gs later, always do New deployment (not edit existing) to apply changes.
